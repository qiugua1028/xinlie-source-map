# 寻源地图 · 快速上手

## 🎯 定位
快消电商行业情报库。服务新猎招聘：拿到品牌 → 找到公司 → 定位猎头目标。

## 🔗 访问
- **Web界面**：http://localhost:8502
- **CLI工具**：`python3 ~/xinlie-source-map/scripts/source_map.py <命令>`
- **数据库**：~/xinlie-source-map/db/source_map.db

---

## 核心功能

### 🔍 搜索（Web / CLI）
```
python3 scripts/source_map.py search 安踏
```
输入品牌名 / 公司名 / 行业关键词，返回品牌→公司映射结果。

### 📦 品牌库
维护品牌基本信息：
- 品牌名、所属公司、行业赛道、主品类、价格定位、标签、备注

### 🏢 公司库
维护公司工商信息：
- 公司全称、法定代表人、注册资本、注册地、成立年份、主营业务、旗下品牌

### ➕ 手动录入
Alan 可通过 Web 界面直接上传新品牌/公司信息。

---

## 数据结构

### 品牌表 (brands)
| 字段 | 说明 |
|------|------|
| brand_name | 品牌名 |
| company_id | 所属公司（关联companies表） |
| industry | 行业：服饰/美妆/电商/食品 |
| category | 主品类 |
| position | 价格定位：高端/中高端/中端/平价 |
| tags | 标签，逗号分隔 |
| notes | 备注 |

### 公司表 (companies)
| 字段 | 说明 |
|------|------|
| company_name | 公司全称 |
| legal_person | 法定代表人 |
| reg_capital | 注册资本 |
| reg_address | 注册地 |
| founded_year | 成立年份 |
| business | 主营业务 |
